## ⚠️ 발견된 주요 한계

### HIGH 우선순위 🔴

#### 1. 객체 리터럴 미지원
```
❌ 현재 (불가능):
let obj = { name: "Alice", age: 25 }

✅ 대안 (가능):
let obj = {}
obj.name = "Alice"
obj.age = 25
```
**영향도**: 심각 - 프레임워크 API 불가

#### 2. Import/Export 미지원
```
❌ 현재 (불가능):
import "./xpress" as xpress

✅ 대안 (불가능):
// 모듈 분할 불가능
// 단일 파일만 가능
```
**영향도**: 심각 - 모듈화 불가능

#### 3. 한글 미지원
```
❌ 현재 (불가능):
println("한글 테스트")

✅ 대안:
println("Korean test")
```
**영향도**: 중간 - 영문으로 제약

#### 4. 블록 주석 미지원
```
❌ 현재 (불가능):
/** 블록 주석 */

✅ 대안:
// 단일 라인 주석만 가능
```
**영향도**: 낮음 - 문서화에만 영향

### MEDIUM 우선순위 🟠

#### 5. 함수 정의 제약
```
❌ 현재 (불가능):
fn test() { ... }

⚠️ 현재 상태 불명확:
실제 문법 검증 필요
```

#### 6. 배열 생성자 부족
```
❌ arr.of(...) 미지원
❌ [1, 2, 3] 리터럴 미지원?
```

## 🔧 수정 전략

### Phase 1: 영문 전환 (즉시)
모든 주석과 문자열을 영문으로 변환:
```fl
// Before (한글)
println("로깅 미들웨어")

// After (영문)
println("Logging middleware")
```

### Phase 2: 모놀리식 구조 (필수)
모듈 분할 포기, 단일 파일에서:
```fl
// 모든 함수를 main.fl에 통합
fn createApp() { ... }
fn createRequest() { ... }
fn createResponse() { ... }
```

### Phase 3: 객체 초기화 변경 (필수)
```fl
// Before (불가능)
let app = { routes: [], port: 3000 }

// After (가능)
let app = {}
app.routes = {}
app.port = 3000
```

### Phase 4: 함수 문법 검증 (필수)
실제 지원하는 fn 문법 확인:
```fl
// 가능한지 확인 필요:
fn test(x) {
  return x + 1
}
```
